Triangle Project

Небольшой проект на Python, который определяет тип треугольника по трём сторонам и вычисляет координаты его вершин для отображения в поле 100x100 px.

Что умеет программа
определяет тип треугольника:
равносторонний
равнобедренный
разносторонний
не треугольник
проверяет корректность введённых данных
вычисляет координаты вершин
масштабирует координаты под размер 100x100
записывает все запросы в лог
Входные данные

Программа принимает 3 строки — длины сторон треугольника.

Пример:

triangle_info("3", "4", "5")
Что возвращает

Тип треугольника и координаты вершин:

('разносторонний', [(x1, y1), (x2, y2), (x3, y3)])

Если из сторон нельзя построить треугольник:

('не треугольник', [(-1, -1), (-1, -1), (-1, -1)])

Если данные введены неправильно:

('', [(-2, -2), (-2, -2), (-2, -2)])
Логирование

Все запросы записываются:

в консоль
в файл triangle.log

Успешные запросы — когда удалось определить тип треугольника.

Неуспешные — ошибки, неправильные данные или невозможность построить треугольник.

Запуск
python main.py
